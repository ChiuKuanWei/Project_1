# 產品良率檢測系統(WinForm c# + SQL Server)

功能需求:

統計各站產品檢測結果，(自動/手動)。

大概流程: 

取得每欄計數->設備連線->不斷詢問是否進行測試->取得測試結果->進行結果統計->告知設備分析處理完畢
